Genesis Marine Intelligence

Multi-sea marine eutrophication, climatology and forecasting workspace.

Source on GitHub

This is the public, citation-ready face of the Genesis Marine workspace. It ships pre-computed time series, monthly climatology, annual variability, summary statistics and 12-month SARIMA forecasts for the parameters most relevant to marine eutrophication monitoring. Data is refreshed weekly by a GitHub Actions pipeline that pulls from Copernicus Marine (BLKSEA, MEDSEA and global biogeochemistry products).

Methodology

  • Data source: Copernicus Marine biogeochemistry products — BLKSEA (~2.5 km), MEDSEA (~4.2 km) and GLOBAL (~0.25°). The basin covered by each sea is configured in config.SEAS.
  • Time series: spatial mean over each sea's bounding box, depth-averaged 0–10 m, monthly resolution.
  • Climatology: calendar-month mean and standard deviation across the published window.
  • Forecast: SARIMA(1,1,1)(1,1,1,12) on the spatial-mean series. Prophet is intentionally skipped on CI to keep the run lightweight and reproducible.
  • Refresh cadence: weekly via GitHub Actions (refresh-data.yml). Deployment to GitHub Pages happens automatically after the refresh job succeeds.
